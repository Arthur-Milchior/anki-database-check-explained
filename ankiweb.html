WARNING: This add-on is still in beta mode. It is supposed to Fix the database. However, as anything updating the database, it may actually hurt it if there is a bug not yet found. Hence, be sure to have a backup of your collection before running «Check Database». Rationale This add-on simultaneously solves three related problems.
